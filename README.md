# Mik - Personal Portfolio

Static, responsive portfolio for Mukesh Kumar, built with semantic HTML, CSS, and minimal vanilla JavaScript.

## Run locally

Open `index.html` in a browser. No build tools or server are required.

## Included assets

- `assets/images/mukesh-kumar.jpg` - hero portrait
- `assets/Mukesh_Kumar_Resume.pdf` - downloadable resume

To replace either item, overwrite the file while retaining its name, or update the corresponding path in `index.html`.

## Responsive approach

The layout uses CSS Grid for desktop sections and collapses to one or two columns at tablet and phone breakpoints. Navigation becomes a keyboard-accessible menu on small screens, and spacing and typography scale with `clamp()` values.
